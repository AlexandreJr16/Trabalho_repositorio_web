Projetos

Conheça meus projetos

TO DO

O "Todo" é uma ferramenta de organização de tarefas que permite ao usuário adicionar, concluir e excluir tarefas em um projeto. Esse recurso é desenvolvido utilizando somente HTML, CSS e JavaScript, o que torna sua implementação simples e prática.

JOGO DA VELHA

O jogo da velha, também conhecido como "Tic Tac Toe", é um jogo projetado para ser jogado por duas pessoas. Neste projeto em particular, o jogo é desenvolvido utilizando HTML, CSS e principalmente JavaScript, proporcionando uma experiência interativa e dinâmica aos usuários.

Gerador de senhas

Como uma forma de testar meus conhecimentos em HTML, CSS e JavaScript, eu decidi criar um gerador de senhas "aleatórias" e o resultado foi uma ferramenta de fácil utilização, sendo necessário apenas um clique para a geração.

Conversor de Temperatura

Durante as aulas de termometria na escola, tive a ideia de criar um conversor de temperatura, utilizando os conhecimentos adquiridos em WEB 1 e Física. Com esse projeto pude ter mais eficiência em alguns exercícios do conteúdo.

Cronômetro

O cronômetro é uma ferramenta desenvolvida em JavaScript, HTML e CSS, utilizada para contagem de tempo.Essa ferramenta é extremamente útil em diversos contextos, como em atividades esportivas, jogos e em processos produtivos que envolvem tempo limitado.

Réplica do site do discord

O Discord é uma plataforma mundialmente famosa, cuja página serviu como referência para a replicação de um site em HTML e CSS neste trabalho. O trabalho não se atentou a responsividade, por isso, pode ser visto pequenas falhas dependo do tamanho da tela.
